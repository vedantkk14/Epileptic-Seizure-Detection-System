About the Project

This project uses advanced Machine Learning techniques (Support Vector Machines with PCA preprocessing) to predict whether a person is likely to experience an epileptic seizure based on EEG signal readings. Our model is trained on a dataset containing 178 features and achieves outstanding accuracy with state-of-the-art performance metrics.

How It Works

Step 1: Upload Data

Upload a CSV file containing a single row with 178 EEG features extracted from brain signal recordings.

Step 2: Data Processing

The system applies standardization and Principal Component Analysis (PCA) transformation to optimize your data for prediction.

Step 3: AI Prediction

Our trained Support Vector Machine model analyzes the processed data and provides instant seizure likelihood assessment.
